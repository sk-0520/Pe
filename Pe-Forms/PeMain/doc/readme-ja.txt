# Pe

## 概要

本ソフトウェアはフリーソフトウェアです。
細かい使用方法等はオンラインヘルプ(https://bitbucket.org/sk_0520/pe/wiki)を参照してください。


## ライセンス

ファイル [<Pe>/doc/license/Pe.txt] を参照。


## 必要環境

 * Windows7(x86/x64)
 * .NET Framework 4.5.1


## インストールとアンインストール

インストールはこの文書を読めてさえいれば必要ありません。
アンインストールは基本的に Pe のディレクトリを削除するだけです。
