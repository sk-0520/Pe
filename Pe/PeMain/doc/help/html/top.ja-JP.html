ヘルプ

概要

本ソフトウェア(以後 Pe)はランチャーが主機能のデスクトップユーティリティです。 Pe は GPLv3 でライセンスされています。

機能

  • アプリケーションデスクトップツールバー型のボタン型ランチャー
  • コマンド型ランチャー
  • ノート(付箋)
  • クリップボード取込・保存
  • テンプレート(定型文)管理
  • ウィンドウ位置保持・復元
  • 拡張子・隠しファイル表示切り替え

必要環境

  • .NET Framework 4.6 以上
  • OS, ストレージ、メモリについては必要な .NET Framweork が動作する環境に準拠

連絡先

  • WEBサイト
  • 開発サイト
  • メール
  • フィードバック(privacy)
