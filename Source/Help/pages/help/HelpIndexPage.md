# 概要

本ソフトウェア(以後 Pe)はランチャーが主機能のデスクトップユーティリティです。 Pe は WTFPL 2 でライセンスされています。

# 必要環境

* [.NET 9](https://dotnet.microsoft.com/download/dotnet/9.0) の動作要件を満たしたハードウェアスペック(環境自体は同梱)
* Windows 10, 11
	* x64 版のみ対応（64 bit 環境で動作）
	* Arm は知らん
