概要

本ソフトウェア(以後 Pe)はランチャーが主機能のデスクトップユーティリティです。 Pe は WTFPL 2 でライセンスされています。

必要環境

  • .NET 7の動作要件を満たしたハードウェアスペック(環境自体は同梱)
  • Windows 10, 11
    • x86 版は 32/64 bit で動作可能
    • x64 版は 64 bit のみで動作可能
    • Arm は知らん
