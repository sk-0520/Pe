NuGetパッケージ

各種コンポーネントは NuGet パッケージを使用して取得しています。

なお、Pe 独自のアセンブリに関しては通常の NuGet パッケージソースではなく myget から取得しています。

VS標準 パッケージソース Feed
o nuget.org https://www.nuget.org/api/v2/
o(*) Microsoft and .NET https://www.nuget.org/api/v2/curated-feeds/microsoftdotnet/
× content-type-text-net https://www.myget.org/F/content-type-text-net/api/v3/index.json

(*)Microsoft and .NET は Visual Studio に標準で設定されていると思うけど設定可能なので準標準くらいの位置づけかと。

パッケージソースcontent-type-text-netは独自フィードであるためダウンロードを許可するよう設定変更してください。
